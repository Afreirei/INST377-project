About This App

Sunny Days is a new weather-aware event planner constructed using React, Supabase, and the Open-Meteo API. Book calendar events and see your event date's forecast in real time. The app dynamically loads your local weather forecast and overlays it on your personal schedule using FullCalendar. Data is written and read in real time via Supabase, providing a seamless and dynamic user experience without a typical server. The project was created to show how real-world APIs and cloud databases can be incorporated into responsive, interactive web apps.
